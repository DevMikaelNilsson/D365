# Complete Tax calculation Feature and update Tax calculation parameters
When you have added the custom column(s) to the Globalization studio Feature, you will need to complete the feature version to save the changes.
Select the current feature and select the draft version, then click on **Change status** and select **Complete**
 
Now the changed draft version is saved and the status has changed to **Completed**.
The next step is to update the Tax calculation parameters to use the new version that you completed.
Access the Tax calculation parameters form.
**URL: Tax -> Setup -> Tax configuration -> Tax calculation parameters**
Select the Feature setup name with the version that you just completed
 
Click to save your changes. The system should now use the new version of your selected feature.
