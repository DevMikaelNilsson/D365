# Overview
This folder contains some information about how to work with the Tax Calculation Service - Globalization Service.

The README files can be read in the following order:
1. Import/Export configuration
2. Add custom column to Applicabillity rules in Globalization Studio
3. Add drop-down menu to column Applicabillity rules in Globalization Studio
4. Extend custom column in Tax calculation service
5. Complete Tax calculation Feature and update Tax calculation parameters
